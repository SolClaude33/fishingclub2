# Penguin Fishing Club v1.2

A Web3 fishing game built with RPG Maker MZ, featuring blockchain integration, wallet connectivity, and interactive minigames.

## Features

- 🎣 Interactive fishing minigame
- 🔗 Abstract Global Wallet integration via Privy
- 💰 Blockchain coinflip minigame
- 💾 Wallet-based save system
- 🎮 RPG Maker MZ game engine

## Web3 Integration

- **Abstract Global Wallet**: Connect using Privy authentication
- **Blockchain Minigames**: Coinflip with Web3 integration
- **Wallet-based Saves**: Game progress tied to wallet address

## Getting Started

1. Open `index.html` in your browser to start the game
2. Click "Connect AGW" to connect your Abstract Global Wallet
3. Start fishing and playing minigames!

## Development

This is a static web game that runs entirely in the browser. No build process is required.

## Deployment

The game is deployed on Vercel as a static site. The `vercel.json` configuration handles the deployment settings.
