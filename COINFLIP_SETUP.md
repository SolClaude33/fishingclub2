# 🪙 Coinflip Plugin Setup

## Instrucciones para activar el Coinflip

### Paso 1: Abrir RPG Maker MZ
1. Abre RPG Maker MZ
2. Carga tu proyecto "Penguin Fishing Club"

### Paso 2: Activar el Plugin
1. Ve a **Tools** → **Plugin Manager**
2. Haz clic en **Add Plugin**
3. Busca y selecciona **CoinflipMinigame.js**
4. Haz clic en **OK**
5. Asegúrate de que el plugin esté **habilitado** (checkbox marcado)
6. Haz clic en **OK** para guardar

### Paso 3: Probar el Coinflip
1. **Guarda tu proyecto** (Ctrl+S)
2. **Ejecuta el juego** (F12 o Play)
3. **Ve al evento coinflip** en el mapa
4. **Interactúa con él** (presiona Enter/Espacio)
5. **¡Debería abrirse el modal del coinflip!**

## 🎮 Cómo funciona

- **Antes**: El evento mostraba "YEAYEAYEA"
- **Ahora**: Se abre un modal con:
  - Moneda dorada (🪙)
  - Botones "Heads" y "Tails"
  - Animación de giro
  - Resultado (ganaste/perdiste)

## 🔧 Si no funciona

1. **Verifica que el plugin esté activado** en Plugin Manager
2. **Reinicia el juego** después de activar el plugin
3. **Asegúrate de que el evento coinflip** tenga el texto "YEAYEAYEA"
